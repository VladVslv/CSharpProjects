Корректный ввод графов (нумерация всех вершин и рёбер начинается с нуля):
1. МАТРИЦА СМЕЖНОСТИ.
На первой строке - количество вершин в графе.
Дальше в каждой строке строки матрицы, где числа разделены пробелами.
Если на позиции (i, j) стоит 1, это значит, что есть ребро от i до j, а если 0 - ребра нет.
Пример:
3
0 1 1
1 0 0
1 0 0

2. МАТРИЦА ИНЦИДЕНТОСТИ.
На первой строке - количество вершин в графе.
Дальше в каждой строке строки матрицы, где числа разделены пробелами.
a) Для неориентированного графа:
Если в столбце на разных позициях i и j стоят 1, это значит, что есть ребро между i и j. На
остальных позициях в столбце стоят 0.
Пример:
3
1 1
1 0
0 1
b) Для ориентированного графа:
Если в столбце на разных позициях i и j стоят -1 и 1 соответственно, это значит, что есть ребро от
i до j. На остальных позициях в столбце стоят 0.
Пример:
3
-1 -1
1 0
0 1

3. СПИСОК СМЕЖНОСТИ.
На первой строке - количество вершин в графе.
Дальше в каждой строке идут строки списка, где числа разделены пробелами.
Если в i-oй строке стоит число j, это значит, что от i до j есть ребро.
Пример для неориентированного графа:
3
2

0
Пример для ориентированного графа:
3
1 2
2
1

4. СПИСОК РЁБЕР.
На первой строке - количество вершин в графе.
На второй строке - количество рёбер в графе.
Дальше в каждой строке стоят пары чисел, разделенные пробелами.
Если в строке написана пара (i, j), то от i до j идёт ребро.
Пример:
3
2
1 2
0 2

Корректный ввод данных:
1. При запросе пути файла, требуется ввести полный путь к текстовому фалу без кавычек.
2. Там, где указано ввести какое-либо число, нужно вводить одно из указанных чисел, после чего
делать перевод строки.

Вывод данных:
1. При выводе графа в файл, он будет перезаписан.
